# PRODUCT PAGE DRAFT

## AI Router MVP

### Short promise
Turn raw conversations into structured files and PowerShell save commands.

### For whom
- people who keep losing context in ChatGPT
- people who use Obsidian as a second brain
- people who want less chaos and fewer manual file moves
- people who think better in conversation than in folders

### What pain it solves
- too many useful chat fragments get lost
- hard to decide where a note belongs
- too much manual folder/file creation
- context becomes heavy and messy
- good ideas stay in chat instead of becoming reusable memory

### What is inside
- router_prompt.txt
- quickstart.md
- examples.md
- launch_router.ps1
- simple product structure for routing text into files

### How to use
1. Take any raw conversation or thought block
2. Paste it into your Router chat
3. Get:
   - destination
   - save block
   - PowerShell command
4. Run the command
5. Keep your system clean without manual sorting

This is an MVP.
It is not magic.
It does not replace thinking.

raw conversation -> AI Router -> structured files -> reusable memory
