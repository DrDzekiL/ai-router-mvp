# TELEGRAM POST DRAFT

I made a small AI Router that turns raw conversations into structured files.

Not an “AI empire”.
Not a trading signal machine.
Not magic.

Just a simple workflow:
raw text -> routing -> save command -> reusable note

I built it because I kept losing useful context in long chats.

The goal is simple:
less chaos,
less manual sorting,
more usable memory.

This is an MVP.
Small, clear, practical.
