# 🚀 Router MVP — Quickstart

## Что это
Система, которая превращает разговоры в структурированные файлы через ИИ.

## Как использовать

### 1. Берёшь любой текст
- диалог с ИИ
- мысли
- торговый разбор

### 2. Вставляешь в чат с Router
Пишешь:

Отсортируй и сохрани это через Hermes Router

### 3. Получаешь ответ
Router выдаёт:
- куда сохранить
- готовый блок
- PowerShell команду

### 4. Копируешь команду в PowerShell
Нажимаешь Enter -> DONE

## Пример логики
Разговор -> Router -> файл -> память

## Важно
- не думай где хранить
- не открывай папки вручную
- не усложняй

Router решает.

## Главное правило
Если это важно -> оно должно быть сохранено
