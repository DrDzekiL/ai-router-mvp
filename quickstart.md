# AI Router MVP — Quickstart

## What it is
A simple workflow that turns raw conversations into structured files with PowerShell save commands.

## How to use

### 1. Take any raw text
Examples:
- a chat fragment
- a market note
- an idea
- a reflection
- a rough draft

### 2. Paste it into your Router chat
Example instruction:

Sort this text and save it through AI Router.

### 3. Get the result
The router should return:
- where it belongs
- a short save-ready block
- a PowerShell command

### 4. Run the command
Press Enter.
The note is saved.

## Why it is useful
- less manual sorting
- less context loss
- clearer file structure
- easier transition from conversation to saved memory

## Important
This is an MVP.
It is not an autonomous agent.
It is a simple routing workflow.
